# HW7 Resume Site

A personal resume website built with HTML and CSS as part of COMP484 Web Development at CSUN.

## Live Site
https://bargavisivaraman.github.io/comp484-hw7/

## Features
- Responsive layout using floats and flexbox
- Google Fonts (Playfair Display + DM Sans)
- Color palette with CSS custom properties
- Profile image with circular crop
- Content wrapper for centered layout
- Sections: About, Skills, Work Experience, Education, Projects, Footer

## Technologies
- HTML5
- CSS3
